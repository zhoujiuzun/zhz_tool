# 📸 zhz工具箱

> 截个图，文字就出来了。需要的话，翻译也一起给你 ✨

一个常驻托盘的 Windows 小工具箱，汇总了我平时用到的一些小工具。缺什么功能，也可以来跟我提哦 (｡･ω･｡)ﾉ

用得最多的是 **OCR** 和**翻译**，国内国外都能用 —— 配好接口，哪个通走哪个，环境变了自动切 (๑•̀ㅂ•́)و✧

## 🚀 跑起来

**打包好了** —— 双击 `软协工具箱.exe` 就行 (´▽`)ノ

启动后缩在 **Win11** 右下角的托盘里 🔽 截图 / 翻译 / 贴图都从**托盘菜单**点。
